## Map Format: Proprietary `.mwm` Files (Not Raw OSM)

Organic Maps does **not** use raw `.osm` or `.pbf` files directly. Instead:

| Component       | Details                                                                                      |
|-----------------|----------------------------------------------------------------------------------------------|
| **File Format** | `.mwm` (Map With Me) – a compressed, indexed binary vector format [[15]]                     |
| **Content**     | Vector geometries + styled attributes + search index + routing graph                         |
| **Advantages**  | Fast rendering, small size (~10-100× smaller than raw OSM), offline-ready                    |
| **License**     | Map data distributed under a custom license (free to use, attribution required) [[12]][[15]] |

> You cannot open `.mwm` files in QGIS or other GIS tools directly. They are optimized for the Organic Maps rendering engine ("Drape") only [[13]].

## How Maps Are Downloaded (User Perspective)

1. **Open Organic Maps** → Tap the **🗺️ Download Maps** icon (bottom toolbar or menu)
2. **Browse by region**: World → Continent → Country → Sub-region (e.g., `Germany → Bavaria → Munich`)
3. **Tap to download**: Files range from ~10 MB (small towns) to ~2 GB (large countries)
4. **Storage location**:
   - **Android**: Internal app storage (`/data/data/app.organicmaps/`) or SD card (configurable in Settings)
   - **iOS**: App sandbox (managed by iOS; not user-accessible)
5. **Updates**: Maps update **biweekly** with fresh OSM snapshots (e.g., "OSM data as of April 4, 2026") [[17]][[19]]

 **No account, no login, no tracking** – downloads are anonymous and privacy-respecting [[12]].

## How Maps Are Rendered & Used Offline

| Feature                     | How It Works                                                                                                   |
|-----------------------------|----------------------------------------------------------------------------------------------------------------|
| **Vector Rendering**        | The "Drape" engine draws roads, buildings, POIs dynamically at any zoom level with smooth rotation/tilt [[13]] |
| **Offline Search**          | A pre-built search index (names, categories, addresses) allows instant local queries without internet          |
| **Offline Routing**         | GraphHopper/OSRM-derived routing graphs are embedded in `.mwm` files for car/bike/walking navigation           |
| **3D Buildings & Contours** | Elevation data and building heights are pre-processed into the vector tiles for visual depth                   |
| **Themes**                  | Day/Night mode swaps style rules (colors, visibility) without re-downloading data                              |

🔹 **All features work 100% offline** after download: search, routing, POI details, voice navigation, bookmarks.

## How Map Updates Work

- Organic Maps **does not sync live OSM edits**. Instead:
  1. OSM data is snapshot periodically (e.g., every 2 weeks)
  2. Servers reprocess snapshots into updated `.mwm` files
  3. App notifies users: *"Map update available"* → tap to re-download region
- **Your edits to OSM** (via iD/JOSM) will appear in Organic Maps only after the next regional reprocessing cycle [[20]].

> Want to contribute? Edit directly on [openstreetmap.org](https://www.openstreetmap.org) – your changes feed all OSM-based apps, including Organic Maps.

```mermaid
graph LR
A[OpenStreetMap Raw Data] --> B[Organic Maps Build Pipeline]

C --> D[HTTPS CDN / Mirrors]
```

- **Build tools**: Proprietary preprocessing pipeline (not public) [[2]]
- **Rendering engine**: "Drape" – custom C++/OpenGL ES vector renderer [[13]]
- **Routing**: Embedded contraction hierarchies for fast A* pathfinding
- **Languages**: App supports 50+ UI languages; map labels use OSM `name:*` tags

## Common Questions

| Question                                    | Answer                                                                                        |
|---------------------------------------------|-----------------------------------------------------------------------------------------------|
| **Can I export `.mwm` files?**              | No – they're app-specific. Export bookmarks/tracks as GPX/KML/GeoJSON instead.                |
| **Can I use Organic Maps data in QGIS?**    | Not directly. Download raw OSM from [Geofabrik](https://download.geofabrik.de/) for GIS work. |
| **How do I get the latest OSM edits?**      | Re-download the region in-app when update notification appears, or wait for biweekly refresh. |
| **Is my location tracked during download?** | No – downloads are anonymous; Organic Maps collects zero telemetry [[12]].                    |
