### Part 1: How to Download OSM Data

| Your Need             | Where to Download                                                                         | Format                     | Size                      |
|-----------------------|-------------------------------------------------------------------------------------------|----------------------------|---------------------------|
| **Country/Region**    | [Geofabrik](https://download.geofabrik.de/) or [BBBike](https://download.bbbike.org/osm/) | `.pbf`, `.shp`, `.geojson` | 10 MB – 5 GB              |
| **Custom area/query** | [Overpass Turbo](https://overpass-turbo.eu/)                                              | `.osm`, `.geojson`, `.csv` | < 1 GB                    |
| **Entire world**      | [Planet OSM](https://planet.openstreetmap.org/)                                           | `.osm.pbf`                 | ~100+ GB (updated weekly) |

**Recommendation:** Always use `.pbf` (Protocolbuffer Binary Format). It's compressed, fast to parse, and supported by almost all OSM tools.

### Part 2: How to Use OSM Data (by scenario)

#### Offline Navigation (Mobile/Desktop)

You don't need to handle raw files. Use apps that bundle OSM data:

- **Organic Maps** (iOS/Android) or **OSMAnd** (iOS/Android/Desktop)
- **Steps:** Install app → Open Settings/Downloads → Pick region → Download → Use fully offline
- Includes routing, POIs, hiking trails, and transit overlays.

#### GIS & Spatial Analysis (QGIS / Python)

- **QGIS:**
  1. Install QGIS (≥3.30 has built-in `.pbf` support via GDAL)
  2. `Layer → Add Layer → Add Vector Layer` → Select `.pbf`
  3. Choose layers (points, lines, multipolygons)
  4. *(Optional)* Install `QuickOSM` plugin for live queries
- **Python:**

  ```python
  import osmnx as ox
  G = ox.graph_from_place("Berlin, Germany", network_type="drive")
  ox.plot_graph(G)
  ```

  (`osmnx` downloads & parses OSM automatically)

#### Web Mapping / Tile Display

- **View online (no download):** Use Leaflet/OpenLayers with OSM tiles:
  `https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png`
- **Self-host tiles:** Use [TileServer GL](https://github.com/maptiler/tileserver-gl) or [Martin](https://github.com/maplibre/martin) with downloaded `.pbf` or `.mbtiles`.

#### Routing & Geocoding Servers

- **Routing:** [OSRM](http://project-osrm.org/), [GraphHopper](https://www.graphhopper.com/), or [Valhalla](https://github.com/valhalla/valhalla)
  - Example (OSRM Docker):

    ```bash
    docker run -v "${PWD}:/data" osrm/osrm-backend osrm-extract -p /opt/car.lua /data/region.pbf
    docker run -v "${PWD}:/data" osrm/osrm-backend osrm-partition /data/region.osrm
    docker run -v "${PWD}:/data" osrm/osrm-backend osrm-customize /data/region.osrm
    docker run -t -i -p 5000:5000 -v "${PWD}:/data" osrm/osrm-backend osrm-routed --algorithm mld /data/region.osrm
    ```

- **Search/Geocoding:** [Nominatim](https://nominatim.org/) (requires PostgreSQL + PostGIS)
